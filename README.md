# AMP Story Ads Preview Tool

## Building

[`yarn`](https://yarnpkg.com) is used for package management.
[Install `yarn`,](https://yarnpkg.com/en/docs/install#mac-stable) then install this project's dependencies through:

```sh
yarn
```

### Developing

```sh
yarn watch
```

### Build & serve

```sh
yarn serve
```
